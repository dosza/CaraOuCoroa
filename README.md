# Cara Ou Coroa

Um aplicativo simples, para você jogar cara ou coroa

Tecnologias utilizadas
---
+	Java
+	Gradle 

Funcionalidades
---
+	Escolha de cara ou coroa
+	Mostra resultado (venceu ou perdeu)
+	Adiciona contador de vitórias e derrotas
